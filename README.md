**Примечание:** Все команды в данной документации описаны для окружения Ubuntu 22.04. Для других операционных систем команды могут отличаться.

## Требования

Перед запуском приложения необходимо установить следующие компоненты:

1. **Docker**
2. **Python 3**
3. **kubectl** (опционально, только для развертывания с Kubernetes/k3s)
4. **k3s** (опционально, только для развертывания с Kubernetes)

## Документация

Проект поддерживает два способа кластеризованного развертывания:

- **[SWARM.md](SWARM.md)** - Развертывание с использованием Docker Swarm
- **[K3S.md](K3S.md)** - Развертывание с использованием Kubernetes (k3s)
